# 记账 App - 部署(06)

## 本地用(现在就能)
- 成品:`expense_app/index.html`
- 双击用浏览器打开就能用。想给别人:直接把这个文件发过去(微信文件传输助手 / AirDrop)。
- 数据存在你这台机器的这个浏览器里,换设备、清浏览器数据会没。

## 发到网上(想让别人也能打开)
- 最简单:打开 **Netlify Drop**(app.netlify.com/drop)或 Cloudflare Pages,把 `index.html` 拖进去,几十秒出一个公网链接,手机扫码就能开。
- 免费,不用买服务器、不用配置。
- 注意:每个人打开是各自的数据(还是存各自浏览器)。要真正多人共享同一份数据,得上后端 —— 那是以后的事(见 01 需求)。

## 现状
离线单文件已完成,随时能用。线上一步可选,想发就拖一下。

## 建议
- 手机上打开后可"添加到主屏幕",用起来就像个真 App(不用上架应用商店)。
