# 记账 App - 需求调研(01)

## 目标用户
我自己,个人日常记账。一个人用,不涉及别人。

## 核心痛点
现在靠聊天软件(给自己发消息)记账,记下来就是一条条流水。**最大的问题是没有统计**:不知道一个月花了多少,更不知道钱都花在哪些地方。

## 典型使用场景
- 花完钱,随手记一笔(多少钱、花在什么上)。
- 过几天或月底,想看看"这段时间钱花哪了、哪一块花得最多"。

## 成功标准
- 记一笔快、不嫌麻烦。
- **能看到统计,一眼知道钱花到哪些地方了** —— 这是最想要的。

## 这次做什么 / 不做什么
**做:**
- 记一笔:金额 + 分到哪一类 + 备注
- 看统计:总共花了多少 + 按分类看钱花哪了
- **设预算 + 超支提醒**:能设一个预算,花超了用颜色警示(比如变红)

**先不做(以后再说):**
- 多人 / 家庭共用
- 云同步(换手机、换电脑也能看)
- 导出报表 / 图表分析
