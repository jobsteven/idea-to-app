# 记账 App - 风格设计(04)

## 整体方向
俏皮可爱的**小猪存钱罐**风:把记账变成"喂小猪攒钱",有成就感、不枯燥。

## 设计流派
圆润扁平 + 手绘感小插画。大圆角、果冻般柔和阴影,轻松不严肃。

## 主题意象 / 吉祥物
**粉红小猪存钱罐**贯穿全程:
- 顶部放一只小猪 + 存钱进度(预算用掉多少 = 进度)
- 金币元素点缀(记一笔像投一枚币进去)
- **小猪带表情(要做)**:按预算状态变脸 —— 没超、结余多 = 开心;快到预算(用掉 80% 以上)= 紧张;超支 = 哭脸
- **空状态**:还没记账时,放一只饿肚子的小猪 + 一句"喂我第一笔吧",引导第一次记账

## 配色
- 主色(猪粉):`#FF8FAB`
- 强调(深粉):`#E75A85`
- 背景(奶粉白):`#FFF5F7`
- 卡片:`#FFFFFF`
- 金币黄(点缀):`#FFC94D`
- 结余绿(正常):`#3FB984`
- 超支红:`#E5484D`
- 文字(暖深):`#4A3B40`

## 字体气质
用手机自带字体,靠**大圆角 + 稍重字重**营造圆润可爱,不额外装字体。

## 质感 / 图形约定
- 圆角:卡片 18px、按钮 12px
- 阴影:柔和、淡,果冻感
- 图标:小猪存钱罐、金币(先用 emoji 或简单 CSS 图形,不画复杂插画)
- 进度条:做成"存钱罐一点点被填满"的感觉
- **动效**:记一笔成功时,金币掉进存钱罐的小动画 + 小猪点一下头,强化"投币攒钱"的感觉
